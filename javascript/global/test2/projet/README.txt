Nom du projet : Jeu du Morpion interactif

Un jeu de morpion à deux joueurs développé en HTML, CSS et JavaScript, avec une interface simple et interactive.

Fonctionnalités principales :

Saisie des noms des joueurs avant de commencer la partie.

Tour par tour : les croix (X) commencent et les ronds (O) suivent.

Insertion dynamique des symboles dans les cases avec animation de zoom.

Détection automatique de victoire ou de match nul.

Affichage d’un message de victoire ou de match nul avec overlay animé.

Réinitialisation du jeu pour recommencer une partie.

Validation des formulaires pour s’assurer que les deux joueurs ont un nom avant de commencer.

Symboles colorés : croix rouges et ronds bleus.

Animations CSS pour l’insertion des symboles.

Overlay discret pour signaler la victoire sans bloquer le jeu.


Ressources utilisé : 

google image, bibliotheques de fonction javascript, css, liveserver, html, navigateur
